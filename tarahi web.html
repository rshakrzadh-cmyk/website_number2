آشنایی با HTML

معرفی این زبان :

HTML یک زبان نشانه‌گذاری است که برای ساخت صفحات وب استفاده می‌شود. در این پروژه سعی شده است ساختار اصلی یک صفحه وب به‌صورت ساده پیاده‌سازی شود.

این صفحه دارای چینش راست‌به‌چپ است تا متن‌های فارسی به‌درستی نمایش داده شوند. استفاده از تگ‌های مختلف HTML باعث می‌شود محتوا معنا‌دارتر شود.

هدف این پروژه آشنایی با تگ‌هایی مانند پاراگراف، جدول، لیست و فرم است و همچنین یادگیری روش‌های مختلف استایل‌دهی با CSS.

  • استفاده از تگ‌های HTML
  • استفاده از CSS Internal,External,Inline
  • نمایش صحیح متن فارسی
